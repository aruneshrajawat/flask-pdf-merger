About PDF Merger

This is a simple and user-friendly PDF Merger web application. The main purpose of this project is to allow users to easily combine multiple PDF files into a single document without installing any extra software.

Key Features:

  • Upload multiple PDF files at once
  • Merge them quickly and download the final document
  • Clean and minimal interface for better user experience
  • Privacy friendly ‚Äì uploaded files are only used for processing and are not stored permanently

This project was built as a learning project to practice web development (frontend + backend) and to understand how file handling and server-side processing works. In the future, I plan to add more features such as PDF splitting, compression, and password protection.
